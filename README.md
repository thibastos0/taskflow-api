# TaskFlow API

Projeto para estudo de CI/CD com Git e automatização de testes.

## 📋 Sobre o Projeto

Este projeto foi desenvolvido seguindo o passo a passo do repositório: [CI-CD](https://github.com/WellingtonRoque/CI-CD/tree/main)

## ✅ Funcionalidades Implementadas

- API REST desenvolvida com Python
- **Testes Unitários**: Dois testes implementados para validação da aplicação
- **CI/CD Pipeline**: Integração contínua e entrega contínua automatizada
- **Deploy em Produção**: Aplicação implantada no Render

## 🚀 Deploy

A aplicação está disponível em produção no Render:
- **URL**: https://api-ci-dsm-cnph.onrender.com/

## 🛠️ Tecnologias Utilizadas

- Python
- Git & GitHub
- Render (Hospedagem)
- CI/CD (Integração Contínua e Entrega Contínua)
